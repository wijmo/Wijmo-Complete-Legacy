This sample shows how to use the column moving feature.

Column moving is performed by dragging a source column header onto a target column header. A column can be inserted either to the right or to the left of the target column, depending on which border the dragged column is dropped closest to. To add a column to a band, its header should be dropped to the center of the band header.
